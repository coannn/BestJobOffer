# Best Job Offer App 

* This app helps user find the best job offer by comparing the current job and job offers.
* User can input the details of the current job and job offers, including base salary, compensation, location, RSU, retirement benefits, etc.
* User can update the detailed information of all jobs.
* The system will evaluate the job details and output a rank of all jobs.
* User can adjust the settings to change the weights for job details based on their needs.
* User can further compare the details of two selected jobs.
